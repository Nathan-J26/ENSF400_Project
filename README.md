# ENSF400_Project

## INSTALLING PROJECT REQUIREMENTS

1. Run `npm i` to install all required React dependencies
2. Run `pip install Flask flask-cors python-dotenv google-genai supabase` to install the required Python dependencies

## HOW TO RUN THE PROJECT

1. Navigate to the `/docreader` folder
2. Before starting the frontend, add the `.env.local` file (provided on D2L) to the `/docreader/frontend` folder
3. Start the frontend by navigating to `/docreader/frontend` and run `npm start`
4. Before starting the backend, add the `.env` file (provided on D2L) to the `/docreader/backend` folder
5. Start the backend by running `server.py`
